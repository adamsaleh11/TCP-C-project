Program Purpose: Regulates access to the book data of a library, where client's requests are fufilled.
The client can also interact with the user end and view the library to check in and out books.

defs.h

Source Files:

client.c

lib.c
server.c

Data Files:
booksData.txt

Compilation Command:
make

Launching Commands:
./server or server
./client or client

Operating Instructions:
Open two terminal windows.
Run the launching command ./server or server in one terminal
Run the launching command ./client or client in the other terminal
